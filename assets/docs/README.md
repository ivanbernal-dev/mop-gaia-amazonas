Carpeta para publicar documentos descargables del MOP.

Estructura sugerida:
- auditoria/proyectos/2021 a 2026: informes de auditoría externa de proyectos.
- auditoria/revisoria-fiscal: informes, dictámenes, certificaciones y comunicaciones de revisoría fiscal.
- auditoria/planes-mejoramiento: planes de mejoramiento y evidencias de cierre.

Después de cargar un archivo, registra su ruta o enlace en `assets/data/auditoria-mop.js`.
